Seminar: Mandelbrot

The Mandelbrot calculation is done concurrently to examplify how a program can be broken into concurrent processes and how (possibly) a multicore computer can speed up the execution of the image.

At the seminar one could introduce distribution and show that the processing could be distributed over several computers.

Building a server

How do we structure our system to be fault tollerant. Which exeptions are trapped, how do we restart processes etc.

A DNS resolver is built to show how a small server can be structured. The decoing of messages uses the bit-syntax which shows to be a powerfull tool when decoding messages.
